# Daniel Verner — Portfolio

Personal portfolio website showcasing my experience, open source contributions, and expertise as a full-stack software engineer.

**Live:** [danielv27.github.io/Portfolio](https://danielv27.github.io/Portfolio)

## Tech Stack

- **Vue 3** with Composition API and TypeScript
- **Vite** for fast development and builds
- **TailwindCSS** + **DaisyUI** for styling
- **@vueuse/motion** for scroll-based animations
- Deployed via **GitHub Pages**

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```

## Project Structure

```
src/
├── components/
│   ├── sections/       # Page sections (Intro, About, Experience, etc.)
│   ├── experience/     # Experience timeline
│   ├── expertise/      # Expertise cards carousel
│   ├── animation/      # Animated text and hover effects
│   ├── nav/            # Navigation bar and social links
│   └── base/           # Reusable UI components
├── assets/             # Images, icons, and SVGs
└── utils/              # Utility functions
```
